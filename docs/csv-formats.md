# CSV Formats

> مهم: الأعمدة يمكن أن تكون بأي ترتيب، لكن **يجب** أن تحتوي الأسماء المطلوبة.

## 1) donations.csv (سجل التبرعات)
الأعمدة المطلوبة:
- `donation_date` : تاريخ/وقت التبرع (مثال: `2026-03-01 13:20:00` أو ISO)
- `amount` : مبلغ التبرع (رقم)
- `referral_code` : كود الإحالة

اختياري:
- `donor_phone`

## 2) boxes.csv (سجل الصناديق)
الأعمدة المطلوبة:
- `created_at` : تاريخ/وقت إنشاء الصندوق
- `box_number` : رقم الصندوق
- `referral_code` : كود الإحالة

اختياري:
- `donor_phone`

## ملاحظات
- يتم حساب `source_row_hash` من القيم (بعد تنقية بسيطة) لمنع تكرار نفس الصف عند إعادة رفع الملف.
- أي صف ناقص قيم مطلوبة سيتم رفضه مع سبب.
